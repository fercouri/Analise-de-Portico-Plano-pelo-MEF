# Analise-de-Portico-Plano-pelo-MEF
Trabalho 1 da disciplina de MAC008  - Introdução ao Método dos Elementos Finitos 

Utilizando a linguagem de programação que melhor lhe convier, faça as devidas adaptações nos códigos fornecidos e resolva o pórtico plano proposto individualmente para cada aluno. Considere que as barras são feitas de aço estrutural (E=200GPa) e com área de seção circular com 3cm de raio. Comece com uma malha minimamente aceitável (justifique) e aumente duas vezes a malha. Compare e discuta os resultados.  As implementações podem ser feitas em qualquer linguagem de programação. Deverão ser entregues os códigos comentados e o relatório detalhado do trabalho, contendo também os gráficos da configuração deformada do pórtico para cada discretização.
